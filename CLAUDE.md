# Shiro Studio Project Context

This project is a modern web landing built with:

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Goals

- Build a high-performance landing page
- Follow clean UI/UX principles
- Keep components reusable and minimal
- Prioritize clarity, spacing, and typography

## Design Source

- The UI is based on a Figma design system
- Always prioritize fidelity to design, but adapt when necessary for better UX

## Development Principles

- Use semantic HTML
- Avoid unnecessary wrappers and divs
- Keep components small and reusable
- Prefer composition over duplication

## Styling

- Use Tailwind CSS
- Follow consistent spacing scale
- Avoid inline styles

## Output Expectations

- Clean, readable code
- Production-ready structure
- No over-engineering
